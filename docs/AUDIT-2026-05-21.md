# SovLens Critical Audit — 2026-05-21

6-agent swarm. Findings cross-cut into ranked plan.

---

## P0 — Ship blockers / security holes

| # | Issue | Where | Fix |
|---|-------|-------|-----|
| 1 | **Arbitrary file read** via `/image?path=…`, `/video`, `/hls`. Any browser tab on user's machine can exfil `~/.ssh/id_rsa`, keychains, anything | `backend/main.py:83,156,281` | Allowlist: resolved `realpath` must be under a registered folder. Reject symlinks. |
| 2 | **CSP disabled** (`csp: null`) | `frontend/src-tauri/tauri.conf.json:25` | Set strict CSP: `default-src 'self'; img-src 'self' http://127.0.0.1:14793 data: blob:; connect-src 'self' http://127.0.0.1:14793; script-src 'self'; style-src 'self' 'unsafe-inline'` |
| 3 | **CORS `*` + credentials** | `backend/main.py:54-60` | Lock to `tauri://localhost`, `https://tauri.localhost`, `http://localhost:3000`. Drop wildcard from video/HLS responses. |
| 4 | **`shell:allow-execute` granted globally** = XSS → RCE | `frontend/src-tauri/capabilities/default.json:17` | Remove. Sidecar uses externalBin, doesn't need it. |
| 5 | **No auth on backend** — any local process can `/add_folder` arbitrary path then read it | `backend/main.py` all routes | Spawn sidecar with random `--token`; frontend sends `Authorization: Bearer`. |
| 6 | **No virtualization** in MediaGrid — 10k items = 10k DOM nodes, gigabytes of full-res thumbnails | `frontend/src/components/MediaGrid.tsx` | `@tanstack/react-virtual` + `?w=256` thumbnail endpoint + `loading="lazy"`. |
| 7 | **Migration loads entire table into RAM** (`.limit(10_000_000).to_list()`) — catastrophic at 1M rows, non-atomic | `backend/core.py:165-196` | Stream via `to_lance().to_batches()`, write new table, atomic swap. |
| 8 | **`tasks_in_progress` race** — concurrent `/add_folder` calls corrupt `progress.json` | `backend/main.py:35`, `ingestion.py:82` | Single-flight ingest mutex + lock around `_save_progress`. |
| 9 | **CLIP ViT-L/14 outdated** — SigLIP-2-base same 768d, 2.5x faster encode, +4-6 pts R@1, 2.3x smaller | `backend/core.py:28` | Swap to `google/siglip2-base-patch16-384`. |
| 10 | **openai-whisper** is reference-slow | `backend/audio_ingest.py:55` | `faster-whisper` + `distil-large-v3` → 4-6x speedup, lower VRAM, VAD filter free. |

## P1 — High impact

| # | Issue | Fix |
|---|-------|-----|
| 11 | Sidecar `_child` dropped → leaks on crash, port conflict on relaunch | Store in app state, kill on `RunEvent::Exit` |
| 12 | No ANN index auto-build trigger after `/add_files` | Call `should_build_index()` post-ingest debounced |
| 13 | `_probe_codecs` runs ffmpeg on every range request | LRU cache on `(path, mtime, size)` |
| 14 | OCR/YOLO not batched — one frame at a time | Batch 16-32 frames per predict call → 3-5x throughput |
| 15 | Hybrid retrieval missing — BM25 + RRF over Whisper/OCR text would dramatically lift recall | LanceDB 0.14 has `full_text_search`; fuse with RRF k=60 |
| 16 | EasyOCR doesn't use MPS on Mac → slow | Swap to PaddleOCR-mobile / RapidOCR (CoreML/DirectML) → 5-8x |
| 17 | Bare excepts swallow errors; `print` instead of `logger` | Replace all with `logger.exception(...)` |
| 18 | No fetch abort on search/pagination → race-condition results | `AbortController` per ref, cancel previous |
| 19 | Status polling runs every 2s when tab hidden | Gate on `document.visibilityState`; or SSE stream |
| 20 | Polling intervals everywhere → SWR/React Query would kill 80% of state code | Adopt one library |
| 21 | No error boundary — uncaught error = white screen | Add `app/error.tsx` + `app/global-error.tsx` |
| 22 | PyInstaller `--onefile` re-extracts 1.5 GB every launch (4-15s) | Switch to `--onedir` + Tauri `externalResource` |
| 23 | `upx=True` triggers Windows Defender heuristic | Disable; gain <5%, hurts trust |
| 24 | PII in `backend.log` (full home paths) | Redactor: `expanduser('~')` → `~` |
| 25 | Ultralytics phones home by default | `YOLO_OFFLINE=1` env at startup |
| 26 | `read_recent_logs` unbounded alloc — 5GB log OOMs shell | Seek to end - max_bytes, read forward |
| 27 | `backend.log` no rotation | Rotate at 10MB, keep .1 |
| 28 | Sidebar `useState` reads localStorage during render — hydration risk despite `suppressHydrationWarning` | Use `useSyncExternalStore` like splash does |
| 29 | `MediaTile` not memo'd, inline `onItemClick` defeats parent memo | `React.memo` + stable `useCallback` |
| 30 | Modal: no focus trap, no Esc handler, no `aria-modal` | Add keydown listener + focus management |
| 31 | LanceDB stores fp32 → 2x disk vs fp16, no quality loss | Schema migration to `pa.float16()` |
| 32 | Scene detection caps at 100 frames/scene on Extreme → static talking-head spam | Cap to 30; raise pHash threshold to 4 |

## P2 — Quality / debt

- Backend: split `core.py` god module → `models/`, `db/`, `search/`, `migrations/`
- `main.py` 950 lines mixing routing + transcode + helpers → extract
- No eval harness (recall unmeasured) — gate model swaps on `R@1/R@5/MRR` over labeled set
- Updater: single pubkey, no rotation path — embed dual keys for transition
- `dirs = "5"` deprecated → `dirs = "6"` or `directories`
- `image` crate default features pull all decoders → restrict to png/jpg/gif/webp/bmp to shrink attack surface
- Magic numbers (`10_000_000` row limits, `1024*1024` chunks) → constants
- `expand_query` is a stub doing nothing → remove dead code
- Untyped JSON parse + `as` casts → `zod` schemas
- Framer-motion for trivial fades — replace with Tailwind keyframes
- AppleScript clipboard escape incomplete (U+2028/9) → use NSPasteboard via objc2 (already a dep)
- HEIC/libheif RCE CVEs in 2024 → pin + `pip-audit` in CI

## Product gaps — biggest gaps vs Apple Photos / immich / Eagle

### Must-have (sprint-1 wins)
1. **Faceted filters** — date, type, folder, duration. Zero filters today.
2. **Timeline browse** — year/month/day grouping. App today = query box, not library.
3. **Find more like this** — already have CLIP vectors; one button.
4. **Search operators** — `type:video`, `before:2023`, `-screenshot`. Cheap.
5. **Favorites + hide-from-search** — bool column in LanceDB. Right-click.
6. **Watch folders** — fsnotify/ReadDirectoryChangesW. Kills "where are my files?" support load.
7. **Cmd-K command palette** — ~150 LoC, huge perceived-modernity win.
8. **Detailed sync progress** — current file, frames/sec, ETA. Today is vague.

### Differentiators
9. Reverse image search (drag image onto search bar)
10. Duplicate finder (CLIP cosine + side-by-side compare)
11. **Privacy dashboard** — index size, files indexed, last outbound request URL+timestamp, "Delete index" red button. Strongest possible trust signal for target user.
12. Drag-out from modal (Tauri drag) + "Reveal in Finder/Explorer"
13. Map view (EXIF GPS, Leaflet + offline tiles, privacy-clean)
14. Index location override in Settings
15. Keyboard shortcuts (`/`, `j/k`, `Space`, `?`)
16. LAN-only mobile PWA viewer with QR pairing

### Strategic
- People clustering (InsightFace, opt-in, off by default)
- Headless CLI + index export/import (laptop ↔ desktop)

## Recommended execution order

**Sprint 1 — Security + ship blockers (1 week)**
P0 #1-5 (auth + allowlist + CSP + CORS), #7 (migration safety), #8 (ingest mutex).
After this: app is no longer "any browser tab can read your filesystem".

**Sprint 2 — UX critical + perf (1 week)**
P0 #6 (virtualization + thumbnails), P1 #11-13 (sidecar lifecycle, ANN auto-build, probe cache), P1 #17-19 (logging, AbortController, visibility gate). Privacy dashboard (product #11).

**Sprint 3 — AI quality (2 weeks)**
Eval harness first. Then faster-whisper, SigLIP-2, fp16, OCR swap, batching, BM25/RRF fusion. Each gated on eval.

**Sprint 4 — Product gaps (2 weeks)**
Filters, timeline browse, find-more-like-this, favorites, watch folders, Cmd-K.
